# 변수를 활용할 수 있는가 여부를 보는 문제

핵심 포인트는 res = res*10 + int(str[i])이라고 봄 

res라는 변수 *10을 통해 반복이 진행됨에 따라 1 *10 + 2 -> 12 * 10 +5 = 125 같은 식으로 뒤에 수를 계속 붙여가는 방식 

```
str = input()
cnt = 0
res = 0



for i in range(0,len(str)-1):
    if str[i].isdigit():
        res = res*10+int(str[i])
        
        
    
for i in range(1,res+1):
    
    if int(res%i) == 0:
        cnt= cnt+1
    

print(res)
print(cnt) 
```
